How to get Joram?

Joram source and distributions are available regularly on the OW2 website, the Joram jars are also available through Maven.

Look at the Download page.

How to start using Joram?

There is a set of running examples in the samples directory of Joram distribution. They are described in the user guide and some of them are subject to an animated tutorial.
