JORAM - Developers' corner

Welcome at the developers' corner

Many people from various origins have contributed to JORAM, and new contributors are always welcome.

A contribution can be a bug fix or the development of a new feature, but also new or improved documentation, suggestions, feedbacks, debugging.

Simply send your ideas, documents and developments (if any) to the joram team.

The following resources are proposed to the advanced users and developers:

  • The road map presents the expected Joram evolutions. It also contains some calls to contributions.
  • The code updates page provides in-depth descriptions of the evolutions of the SVN code base.
  • The SVN Forge page allows you to get JORAM through SVN, browse the updates, differences, etc.
